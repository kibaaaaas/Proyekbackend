Panduan Penggunaan

Cara menjalankan sistem face recognition

Langkah 1

Daftar Wajah

Jalankan face_recognizer.py menu 1, ikuti petunjuk daftar wajah dengan kamera.

Langkah 2

Latih Model

Menu 2 untuk melatih model face recognition dari sampel wajah yang sudah diambil.

Langkah 3

Jalankan Akses

Menu 3 untuk deteksi real-time. Wajah yang dikenal otomatis tercatat di dashboard.

Langkah 4

Cek Dashboard

Buka halaman ini untuk melihat log kehadiran, export Excel, atau hapus data.
